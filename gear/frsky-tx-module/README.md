# FrSky TX Module

External TX module for the Tango 2. Used exclusively for the Mobula 7 HD, which has a built-in FrSky SPI receiver (not ELRS).

## Notes

- Installed in Tango 2 external module bay
- Dedicated to Mobula 7 HD — no other fleet quads use FrSky
