# Armattan Beaver

5" freestyle with GoPro mount. Day flyer — OG DJI cam for best possible image quality in good light. Great quad.

## Notes

- OG DJI cam (not Polar) — excellent daytime image quality, poor low light
- Flies with GoPro Hero 7
- OSD running NTSC passthrough mode (not canvas/MSP DisplayPort) — older setup, candidate for update
- DSHOT600, bidirectional enabled
