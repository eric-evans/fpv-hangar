# GepRC Phantom

2.5" freestyle toothpick. One of the best quads in the fleet — stock tune flies exceptionally well. Analog. 3S.

## Notes

- **TBS Crossfire RX** — one of the few non-ELRS quads in the fleet; uses Tango 2's built-in Crossfire (not the ELRS adapter)
- Came box stock, only change is props — running biblades instead of stock
- Stock tune is excellent — do not touch
- Analog camera — image quality is a limitation
- Channel map is default AETR (not fleet standard TAER1234) — intentionally left as-is to preserve stock config
- Mode mapping non-standard: ANGLE on AUX3 (not AUX4), no HORIZON — left as-is
- VTX: Race Band ch7 (5880MHz) ✓
